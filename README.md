# FIRS Spring Boot Backend (Runnable)

## Requirements
- Java 17+
- Maven
- MySQL (XAMPP)

## Setup
1. Start MySQL from XAMPP
2. Import database:
   - Open phpMyAdmin
   - Import database/firs_db.sql

## Run
mvn spring-boot:run

Server: http://localhost:8080

## APIs
POST /api/login
GET  /api/products
POST /api/orders

## Frontend
HTML files are empty placeholders. Replace with your files.
